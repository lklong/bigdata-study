# 大数据集群日常巡检手册

> 每日/每周/每月必做巡检项，覆盖所有 14 个组件

---

## 一、每日巡检（5 分钟）

### HDFS
- [ ] NameNode Web UI 正常（`http://nn:9870`）
- [ ] `hdfs dfsadmin -report` 无 Dead DataNode
- [ ] Under-replicated blocks = 0 或在减少
- [ ] 磁盘使用率 < 80%

### YARN
- [ ] ResourceManager Web UI 正常（`http://rm:8088`）
- [ ] 无 UNHEALTHY NodeManager
- [ ] 无长时间 PENDING 的应用（> 30 分钟）
- [ ] 队列资源使用率正常（无队列 > 95%）

### Hive
- [ ] HiveServer2 进程存活
- [ ] `beeline -u "jdbc:hive2://hs2:10000" -e "show databases"` 成功
- [ ] HMS 进程存活 + 底层 DB 连接正常

### Spark/Kyuubi
- [ ] Kyuubi Server 进程存活
- [ ] 无异常终止的 Engine
- [ ] Spark History Server 可访问

### Flink
- [ ] 所有流作业 RUNNING 状态
- [ ] Checkpoint 成功率 > 95%
- [ ] 消费延迟在可接受范围

### ZooKeeper
- [ ] `echo ruok | nc zk:2181` 返回 `imok`（3 个节点都检查）
- [ ] `echo mntr | nc zk:2181` 的 `zk_outstanding_requests` < 10

### Kerberos
- [ ] KDC 进程存活
- [ ] `kinit -kt keytab principal` 成功

## 二、每周巡检（30 分钟）

### HDFS
- [ ] `hdfs fsck / -files -blocks` 无 Corrupt/Missing blocks
- [ ] `hdfs balancer -threshold 10` 检查均衡度
- [ ] 快照空间检查

### YARN
- [ ] 各队列资源使用趋势
- [ ] Container 失败率趋势
- [ ] NM 日志目录大小

### Hive
- [ ] ACID 表 Compaction 队列（无大量积压）
- [ ] 慢查询 Top 10 分析
- [ ] HMS 底层 DB 慢查询

### Iceberg/Amoro
- [ ] `expire_snapshots` 执行（清理旧快照）
- [ ] `remove_orphan_files` 执行
- [ ] 小文件分区数检查（单分区 > 1000 文件告警）
- [ ] Amoro Optimizer 状态正常

### Flink
- [ ] Checkpoint 大小趋势（是否持续增长）
- [ ] State 大小趋势
- [ ] 各作业消费延迟趋势

### ZooKeeper
- [ ] `autopurge` 运行正常（旧快照已清理）
- [ ] Watcher 数量趋势（不应无限增长）
- [ ] 各组件的 Session 数量

### 安全
- [ ] Kerberos Keytab 有效期检查（< 30 天需轮换）
- [ ] Ranger 策略审计异常访问
- [ ] LDAP 用户同步正常

---

## 三、每月巡检（2 小时）

### 容量规划
- [ ] HDFS 存储增长趋势 → 预测何时达到 80%
- [ ] YARN 资源利用率 → 是否需要扩容
- [ ] 各队列资源分配是否合理

### 性能基线
- [ ] 关键 ETL 任务耗时趋势（对比上月）
- [ ] NameNode RPC 延迟趋势
- [ ] HS2 查询响应时间 P99

### 安全审计
- [ ] Ranger 审计日志异常访问模式
- [ ] Kerberos 认证失败事件统计
- [ ] 权限策略变更 review

### 版本和补丁
- [ ] 各组件是否有安全补丁未打
- [ ] JDK 版本是否需要升级
- [ ] 操作系统内核是否需要更新

## 四、巡检自动化脚本

```bash
#!/bin/bash
# daily_check.sh — 每日巡检自动化
echo "=== HDFS ==="
hdfs dfsadmin -report 2>/dev/null | grep -E "^(Dead|Under|Missing)"
echo "=== YARN ==="
yarn node -list -all 2>/dev/null | grep -c UNHEALTHY
echo "=== ZK ==="
for zk in zk1 zk2 zk3; do echo "$zk: $(echo ruok | nc $zk 2181 2>/dev/null)"; done
echo "=== HS2 ==="
beeline -u "jdbc:hive2://hs2:10000" -e "select 1" 2>/dev/null && echo "OK" || echo "FAIL"
echo "=== Flink ==="
curl -s http://flink-jm:8081/jobs/overview | python3 -c "import sys,json; jobs=json.load(sys.stdin)['jobs']; print(f'{len([j for j in jobs if j[\"state\"]==\"RUNNING\"])} running, {len([j for j in jobs if j[\"state\"]!=\"RUNNING\"])} not running')"
```
