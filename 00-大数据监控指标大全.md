## 采集方案

```
推荐架构:
  JMX Exporter → Prometheus → Grafana (可视化)
                            → AlertManager (告警)

或:
  JMX → Telegraf → InfluxDB → Grafana

关键:
  - 所有 JVM 进程开 JMX: -Dcom.sun.management.jmxremote.port=xxxx
  - Prometheus 抓取间隔: 15s
  - 告警规则: 按上面阈值配置
```
